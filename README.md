# Payslip Calculator

A Python application that calculates staff wages based on hours worked, shift dates, and job position/role.

## What it does

- Calculates wage totals from time worked, shift dates, and staff position/role rates.
- Built to handle real-world payroll calculation logic for a care home staffing context.

## Tools & Stack

- **Python**

## Files

- `payslip cal.py` — main application logic
- `calculations.py` — wage calculation functions
- `Westfield Payslip Calculator.zip` — packaged project files

## Status

Individual project. Built to solve a real, practical payroll calculation need.
